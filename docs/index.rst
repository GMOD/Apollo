Welcome to Web Apollo's documentation!
======================================

Web Apollo is an instantaneous, collaborative, genome annotation editor. 
It uses a Java web application / database backend and a Javascript client that
runs in a web browser as a JBrowse plugin.

For general information on WebApollo, go to:
`http://genomearchitect.org/ <http://genomearchitect.org>`__

For more information on JBrowse, visit: `http://jbrowse.org <http://jbrowse.org>`__
